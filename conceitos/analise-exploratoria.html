Análise exploratória de dados

Definição simples: Investigação inicial de dados para compreender distribuição, padrões, inconsistências, hipóteses e limites.

Explicação didática

Investigação inicial de dados para compreender distribuição, padrões, inconsistências, hipóteses e limites. No portal unesp.IA, este conceito é trabalhado com exemplos práticos, atividades guiadas, documentação e cuidado com dados pessoais.

Exemplos de aplicação

Educação: organizar inscrições, presenças, relatórios e materiais didáticos.

Dados: preparar bases, validar campos, gerar gráficos e indicadores.

IA: alimentar automações, modelos e análises com dados confiáveis.

Riscos e cuidados

Verifique qualidade, privacidade, finalidade, autorização, revisão humana e documentação das decisões.
