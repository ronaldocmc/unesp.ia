# Portal Didático UNESP.IA - Versão 3

Material didático navegável para participantes, organizado no padrão da Coleção Editorial UNESP.IA.

## Conteúdo
- Módulos completos extraídos do material-base.
- Unidades linkadas individualmente.
- Enciclopédia de conceitos com exemplos didáticos.
- Biblioteca de ferramentas.
- Laboratórios práticos por unidade.
- Caderno do Participante.
- Guia do Instrutor.
- Caderno de Atividades.
- Banco de Estudos de Caso.
- Banco de Prompts.
- Banco de Slides.
- Banco de Figuras e Infográficos.
- Personagens lúdicos da coleção.

## Banco visual
Inclui 1050 arquivos SVG originais:
- 300 infográficos
- 200 fluxogramas
- 150 mapas mentais
- 400 ilustrações didáticas

## Como usar
Abrir `index.html` no navegador.
