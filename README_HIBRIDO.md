# UNESP.IA Portal Híbrido

Esta versão mantém o layout e o conteúdo da versão `unesp_ia_portal_v3` nas páginas principais, especialmente `index.html`, e substitui as páginas dos módulos pelo padrão visual do `UNESP_IA_PORTAL_v5_Sprint1/modulos/m1.html`, com conteúdo expandido/completo dos módulos.

Abra `index.html` para iniciar. Os links dos módulos apontam para `modulos/m1.html` ... `modulos/m10.html`.
