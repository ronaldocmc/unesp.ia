Teachable Machine

Treinamento de modelos simples de classificação com imagem, som ou pose sem programação avançada.

Tipo

Ensino/Aprendizagem

Site oficial

https://teachablemachine.withgoogle.com/

Limitações gratuitas

Recursos gratuitos podem limitar quantidade de gerações, turmas, integrações, exportações e histórico.

Limite diário/cota no anexo

Anexo: não indica limite gratuito específico.

Uso de dados (Treino)

Dados de aulas e estudantes podem ser processados. Use exemplos fictícios ou dados anonimizados.

Entrada

Webcam, microfone, imagens, sons ou exemplos de pose.

Saída

Modelo treinado, predições, exportações e demonstrações interativas.

Funcionalidades relacionadas ao curso

  • Demonstrar aprendizado de máquina
  • Comparar dados de treino
  • Discutir viés e generalização

Exemplos de atividade

  1. Treine um classificador simples com imagens autorizadas.

Cuidados de uso

Modelo simples não prova funcionamento em contexto real.
