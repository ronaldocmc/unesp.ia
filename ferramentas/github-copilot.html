GitHub Copilot

Assistente de programação da GitHub que sugere código, explica trechos e apoia fluxos de desenvolvimento.

Tipo

Código

Site oficial

https://github.com/features/copilot

Limitações gratuitas

Planos gratuitos podem limitar uso, contexto, integrações com IDE, modelos avançados ou recursos corporativos.

Limite diário/cota no anexo

Anexo: cerca de 2.000 completions e 50 requisições no limite indicado.

Uso de dados (Treino)

Trechos de código e prompts (instruções para a IA) podem ser enviados ao serviço. Evite chaves, senhas, dados privados e código confidencial.

Entrada

Código, comentários, arquivos de projeto e instruções.

Saída

Sugestões de código, explicações, testes e refatorações.

Funcionalidades relacionadas ao curso

  • Acelerar protótipos
  • Explicar trechos de código
  • Sugerir testes

Exemplos de atividade

  1. Peça testes para uma função simples e revise cada caso sugerido.

Cuidados de uso

Não aceite código sem entender, testar e verificar segurança.
